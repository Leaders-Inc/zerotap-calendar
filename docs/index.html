プライバシーポリシー

ZeroTapCalendar - AI日程調整＆予定管理

最終更新日: 2026年3月29日

📋 はじめに

Leaders Incorporated（以下「当社」）は、ZeroTapCalendar（以下「本アプリ」）をご利用いただくユーザーの皆さまのプライバシーを尊重し、個人情報の保護に努めます。 本プライバシーポリシーは、本アプリが収集する情報、その使用方法、およびユーザーの権利について説明します。

🔍 収集する情報

本アプリは、以下の種類のデータを収集・利用します。

1. 音声データ

音声入力機能をご利用の際、マイクから取得した音声をAppleの音声認識サービス（Speech Framework）を用いてテキストに変換します。 音声データは端末上で処理され、当社のサーバーには保存されません。

2. 画像データ

カメラまたは写真ライブラリから取得した画像は、端末上のOCR（文字認識）処理にのみ使用されます。 画像データは当社のサーバーには送信・保存されません。

3. テキストデータ（AI処理）

予定を自動生成する目的で、ユーザーが入力した音声やテキスト（連絡先に入っている名前が含まれる場合がある）をOpenAIのAPIに送信します。 送信されるテキストには個人を特定する情報が含まれる可能性がありますが、当社はこのデータを保存しません。 OpenAI社のデータ取り扱いについては、OpenAIのプライバシーポリシーをご参照ください。

4. 連絡先情報

カレンダーイベントの参加者アイコン表示のために、端末の連絡先へアクセスします。 連絡先のデータ自体はサーバーにアップロードしておらず、端末のローカル上で参加者のアイコンを表示するためだけに使用します。

5. カレンダーデータ

本アプリはiOS標準カレンダー（EventKit）にアクセスし、予定の読み取り・書き込み・削除を行います。 カレンダーデータは当社のサーバーには送信されません。

6. 購入情報

アプリ内課金（サブスクリプション）の購入処理はAppleのStoreKitを通じて行われます。 当社はクレジットカード番号等の支払い情報に一切アクセスできません。

🔗 サードパーティサービス

本アプリは以下のサードパーティサービスを利用しています。

サービス 提供元 目的
Google AdMob Google LLC バナー広告およびリワード広告の配信
OpenAI API OpenAI, Inc. テキストからのカレンダー予定解析
Apple StoreKit Apple Inc. アプリ内サブスクリプション処理

広告について

本アプリはGoogle AdMobを使用して広告を表示します。AdMobはパーソナライズド広告を配信するために、デバイスの広告識別子（IDFA）を使用する場合があります。 ユーザーはiOSの設定から「Appにトラッキングを要求する」を無効にすることで、広告のパーソナライズを拒否できます。 ZeroTap Proにアップグレードすると、すべての広告が非表示になります。

Google AdMobのプライバシーポリシーについては、Googleのプライバシーポリシーをご参照ください。

🎯 データの利用目的

収集したデータは、以下の目的でのみ使用されます。

  • 音声・画像からカレンダー予定を生成するための AI 解析処理
  • カレンダーイベントの作成・表示・編集・削除
  • 参加者の連絡先情報（アイコン・氏名）の表示
  • 広告の配信（無料ユーザーのみ）
  • アプリ内サブスクリプションの管理
  • 予定前のリマインダー通知の送信

🔒 データの保存と共有

  • 音声データ、画像データ、連絡先データ、カレンダーデータは端末上でのみ処理され、当社のサーバーには保存されません。
  • AI解析のために送信されたテキストデータは、当社のサーバーには保存されません。OpenAI社のデータ保持ポリシーに従います。
  • 当社はユーザーの個人情報を第三者に販売、貸与、またはその他の方法で提供することはありません（法令に基づく場合を除く）。
  • 広告配信に関するデータは、Google社のプライバシーポリシーに基づいて処理されます。

👤 ユーザーの権利

ユーザーは以下の権利を有します。

  • アクセス権限の管理 — iOSの設定から、マイク・カメラ・写真・カレンダー・連絡先・通知の各権限をいつでも変更できます。
  • 広告トラッキングの拒否 — iOSの「設定」>「プライバシーとセキュリティ」>「トラッキング」から広告トラッキングを無効にできます。
  • データの削除 — 本アプリをアンイストールすることで、端末上のすべてのアプリデータが削除されます。
  • サブスクリプションの管理 — iOSの「設定」>「Apple ID」>「サブスクリプション」からいつでも解約できます。

🧒 子どものプライバシー

本アプリは13歳未満の子どもを対象としていません。 当社は13歳未満の子どもから意図的に個人情報を収集することはありません。 万が一、13歳未満のお子さまの個人情報を収集したことが判明した場合は、速やかに削除いたします。

📝 プライバシーポリシーの変更

当社は必要に応じて本プライバシーポリシーを更新する場合があります。 重要な変更がある場合は、本ページにて通知いたします。 変更後のプライバシーポリシーは、本ページに掲載した時点から効力を生じるものとします。
